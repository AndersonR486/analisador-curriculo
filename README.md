ATS Resume Analyzer

Sistema simples de análise de currículos em PDF com foco em compatibilidade para vagas de emprego. O projeto realiza leitura automática de currículos, identifica palavras-chave, calcula score ATS e gera gráficos de conhecimentos utilizando JavaScript.

Sobre o Projeto

O ATS Resume Analyzer é um mini projeto Front-End desenvolvido com HTML, CSS e JavaScript puro. O objetivo é simular uma análise básica de currículos semelhante aos sistemas ATS utilizados por recrutadores.

O usuário pode:

Fazer upload de um currículo em PDF Inserir uma descrição de vaga Comparar habilidades do currículo com os requisitos da vaga Visualizar compatibilidade em porcentagem Receber sugestões de melhoria Visualizar gráficos de conhecimentos técnicos Funcionalidades Upload de currículo em PDF Extração automática de texto com PDF.js Comparação de palavras-chave Score ATS automático Compatibilidade com vaga Gráfico Radar de conhecimentos Dashboard visual Interface responsiva Drag and Drop para upload Sugestões de melhoria para currículo Sistema básico de análise semântica Tecnologias Utilizadas HTML5 Semântico CSS3 JavaScript PDF.js Chart.js
